# docker save

<!---MARKER_GEN_START-->
Save one or more images to a tar archive (streamed to STDOUT by default)

### Aliases

`docker image save`, `docker save`

### Options

| Name             | Type     | Default | Description                                                                                                                                                                                  |
|:-----------------|:---------|:--------|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| `-o`, `--output` | `string` |         | Write to a file, instead of STDOUT                                                                                                                                                           |
| `--platform`     | `string` |         | Pick a single-platform to be saved if the image is multi-platform.<br>Full multi-platform image will be saved if not specified.<br><br>Format: os[/arch[/variant]]<br>Example: `linux/amd64` |
