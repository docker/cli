# docker whoami

<!---MARKER_GEN_START-->
Display the username of the currently logged in user.
Defaults to Docker Hub if no server is specified.

### Options

| Name    | Type   | Default | Description                                        |
|:--------|:-------|:--------|:---------------------------------------------------|
| `--all` | `bool` |         | Display usernames for all authenticated registries |
